# Human Challenge

A collection of HUMAN Security (PerimeterX) challenge page customizations and demonstrations. Provides custom-branded challenge UI implementations for bot detection flows.

## Contents

| File | Description |
|------|-------------|
| `human-challenge.js` | Main HUMAN challenge configuration with custom branding |
| `human-challenge-ico.js` | Icon variant of the challenge |
| `human-challenge-partner.js` | Partner-branded challenge variant |
| `human-challenge-pfj.js` | PFJ-branded challenge variant |
| `human-challenge-custom-action.js` | Custom action challenge |
| `human-challenge-o.js` | Additional challenge variant |
| `texas-challenge.js` | Texas-themed challenge variant |
| `bhenning.js` / `bhenning.css` | Personal branding assets |
| `client-side.js` | Client-side challenge logic |
| `cookie-sync.js` | Cookie synchronization utility |
| `index.html` | Demo page for testing challenges |

## Demo

The challenge pages are hosted at:

- `https://henninb.github.io/human-challenge/index.html`
- `https://henninb.github.io/human-challenge/human-challenge.js`

## Customization

Each `human-challenge-*.js` file configures the HUMAN Security challenge widget (`window._PX*`) with custom:

- Header text and font size
- Logo image
- Message text and styling
- Button labels
- Color scheme
